## Overview

This research investigates reasoning-layer vulnerabilities in autonomous AI-driven payment systems through adversarial prompt injection attacks targeting Google’s Agent Payments Protocol (AP2).

The work focuses on how malicious contextual instructions can manipulate agent reasoning despite cryptographic protections at the execution layer.

## Contributions

- Designed Branded Whisper and Vault Whisper attacks
- Built experimental AP2 environment using A2A and MCP
- Conducted systematic red-teaming evaluations
- Proposed mitigation strategies for safer agentic commerce systems

## Paper

[Whispers of Wealth: Red-Teaming Google's Agent Payments Protocol](https://arxiv.org/abs/2601.22569)
