- Spring Boot
- MySQL
- JWT Authentication
- Next.js

Implemented:
- Book borrowing system
- Fine management
- Transaction tracking
- Role-based access control
- RESTful APIs

**Technical Areas:**  
Database Systems, Full-Stack Development, REST APIs, Authentication
